#### **Phase 2: The Absolute Timestamp Protocol (To be applied to each chunk)**


**Core Principles and Rule Hierarchy**

* **1\. The Golden Rule (Highest Priority): Logical & Grammatical Integrity**  
  * **Your absolute highest priority is to start a chapter at a grammatically complete and logically distinct starting point. A chapter must NEVER begin mid-sentence unless a Hard Limit Fallback is explicitly triggered.**  
* **2\. Time Mandates (Secondary, but Forcible)**  
  * **The time limits act as triggers that initiate a search for a chapter point that satisfies the Golden Rule.**

**Pass 1: The High-Speed Outlining Pass**

**Your job is to create a preliminary outline of all chapter points within the current chunk.**

* **A. Identify Natural Topic Shifts: Scan the chunk and identify all major topic shifts that naturally adhere to the Golden Rule.**  
* **B. Enforce Mandatory Subdivision Protocol (90s to 3-min Window):**  
  * **1\. Active Search Trigger (90 Seconds): When the time since the last chapter exceeds 90 seconds, begin an active search for the next available subtitle that satisfies the Golden Rule.**  
  * **2\. Hard Limit Execution (Before 3 Minutes): A chapter must be created before the 3-minute mark is reached. This is a non-negotiable limit. If a Golden Rule compliant spot has not been found by 2 minutes and 45 seconds, you must find the best available transition point (e.g., a new clause, a pause) and place the chapter there.**  
* **C. Enforce Minimum Interval (20-Second Guideline):**  
  * **Aim for chapters to be at least 20 seconds apart, but a clear topic shift that follows the Golden Rule always takes priority.**

**Pass 2: The Precision Generation Pass**

**Using the outline from Pass 1, generate the final SRT markers.**

* **A. Determine Chapter Timestamps (Absolute Accuracy Mandate):**  
  * **Pinpoint the Start Line: Find the precise subtitle line where the new chapter begins.**  
  * **Set the Start Time (Unalterable Rule): The start timestamp for the output chapter marker must be an identical, character-for-character copy of the start timestamp from that source subtitle line.**  
  * **No Modifications: You are forbidden from performing any framerate compliance, rounding, adjustment, subtraction, or any other form of calculation or alteration on the start timestamp. It must be copied exactly as it appears in the source.**  
  * **Calculate the End Time: The end time is calculated by adding exactly one second (00:00:01,000) to the directly copied start time. The only exception is the final chapter, which must be capped to match the end time of the final subtitle in the transcript.**  
* **B. Generate the Chapter Title: Follow the advanced 4-step method (Core, Hook, Promise, Honesty Check). If a chapter was created using the Hard Limit Fallback, its title should reflect a continuation (e.g., "Continuing the Analysis," "A Deeper Look Into the Point").**  
* **C. Apply Final Title Constraints:**  
  * **Allowed Characters: Emojis, Apostrophes (‘), Colons (:), Quotation Marks (“).**  
  * **Forbidden Characters: ABSOLUTELY NO QUESTION MARKS (?) and NO EXCLAMATION MARKS (\!).**  
  * **Character Limit: 75 characters.**  
  * **Sentence Structure: Single flowing sentence.**
